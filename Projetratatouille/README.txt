Bienvenue dans le README de notre premier projet fictif réalisé avec la Wild Code School!

Ce projet a pour but de construire un site Web statique avec pour bases de langages le HTML et le CSS.

il s'agit d'un site internet pour une chaîne de restaurant nommé Ratatouille.

Les clients (fictifs donc) souhaitent un site à la fois informatif pour :
 - expliquer leur concept
 - savoir où les trouver
 - avoir une partie pour les personnes qui souhaitent ouvrir leur franchises Ratatouille dans le but de s'étendre.

 Nous sommes libres de proposer un projet qui répondra à ces attentes et travaillons en binôme.
